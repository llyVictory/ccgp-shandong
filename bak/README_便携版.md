# 📦 山东政府采购爬虫 - 便携版

## 🚀 3步快速开始

### 1. 系统要求

- ✅ Windows 10/11
- ✅ Python 3.8+ （[点击下载](https://www.python.org/downloads/)）
  - ⚠️ **安装时必须勾选 "Add Python to PATH"**
- ✅ Google Chrome （[点击下载](https://www.google.com/chrome/)）

### 2. 首次安装

双击运行 `首次安装.bat`，等待3-5分钟完成依赖安装。

### 3. 启动使用

双击运行 `启动爬虫.bat`，浏览器自动打开控制台。
